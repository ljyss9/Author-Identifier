说给左耳
小耳朵（李珥） }
　　温柔坚毅的女生。一直是听话的乖孩子,个性里却有着执着和爱的倔强。在情窦初开的年纪，她喜欢上了好男生许弋，为了这份单恋付出了很多，为了许弋，即使被打得头破血流也在所不惜，因为她知道，吧啦会保护她的。她不顾众人的反对,和坏女孩吧啦成为了好朋友。吧啦在离开人世说的最后一句话就是“谢谢你小耳朵”。最终，小耳朵爱上了张漾，找到了真正的归宿。她对许弋说：我心居然再无波澜，好像年少的坚贞只是一场梦。她对张漾说：跟着你，在哪儿，做什么，都好。她对吧啦说：你是我的好朋友。请记住，这个美好的女孩。
　　PS：喜欢纯牛奶，喜欢粉红色T恤。喜欢在清晨的第一缕阳光里拉开窗帘。
　　乖乖女的外表下，隐藏着深度倔强。
　　张漾失聪的左耳耐心等待，终于听到了，最动听的誓言。
　　“跟着你，在哪儿，做什么，都好。”

{张漾 }
　　有一点坏，有一点酷，非常英俊的男生，吧啦的爱人。曾经因为自己的母亲抛弃自己嫁给许弋的父亲而对许弋满怀仇恨，以至利用吧啦报复许弋，想使其身败名裂。但是他也真心爱着吧啦，吧啦的死让他深深愧疚。后来，父亲告诉了他“母亲改嫁”的真相……张漾的心里终于没有了仇恨。爱上小耳朵，爱上那个单纯的女孩。不惜一切，要让小耳朵幸福。最后，他在许弋报复蒋皎的时候挺身而出，从着火的酒吧救出了很多的人，自己短暂失忆，最终回到了小耳朵的身边。
　　PS：天中历史上最帅的男生，可能也是最骄傲的一个。

左耳
年少的骄傲，总会伤害最爱的人，只可惜，张漾当时不懂得。
　　当他终于放下仇恨，为亲人冲向火海的那刻，才是真正长大了。
　　所以，才拥有了重新幸福的权利。
　　“甜言蜜语，一定要说给左耳听。”

{吧啦 }
　　张扬，放肆，敢爱敢恨，向日葵一样妖艳的女生，喜欢绿色眼影――LANCOME COLOR FOCUS , DEEP FOREST（深深的丛林），妖艳不可方物。爱上了帅气的张漾，不惜为他去欺骗许弋。结识了乖乖女小耳朵，对生命中唯一的好朋友心怀感激。吧啦最后死于车祸，一生唯一的好朋友就是李珥。
　　PS：吧啦，她的出现曾经轰动整个天中。
　　她摧毁的，不只是许弋的乖孩子生涯。
　　连带的，还有蒋皎的骄傲，张漾的不羁，还有李珥纯白的信仰。
　　她的消失，像一颗种子被埋下。

{许弋（许帅） }
　　纯洁的好孩子，像是被放逐凡间的天使，但是吧啦的出现改变了他的生活，吧啦死后，他变得不相信爱情，自己的家庭也迭遭变故。他短暂放任过自己，却因为另一个女孩夏米米的出现，找回纯真的初恋感觉。但是夏米米最终被蒋皎的经纪人吴明明害死，可惜许弋以为夏米米是被蒋皎害死。所以许弋不顾一切找蒋皎同归于尽，在他离开这个世界以前，在小耳朵的博客里写下：小百合，我一直记得爱过你。
　　PS：其实，许弋一直都没有变坏。他只是一直没学会如何去爱。

左耳
对于这个男生，我们总是不懂得如何责备。
　　还是请记得，他为了爱情奋不顾身的模样。
　　还记得，他也曾经深深地，爱过你们。
　　可怜的孩子，我们永远也回不去
　　“小百合，我一直记得爱过你。”

{夏米米　夏吉吉 }
　　夏米米夏吉吉：长相很相似的两姐妹，性格却全然不同。她们的父亲是怀才不遇的画家，后来落魄而死。米米为了生存，为了姐姐，可以利用别人，不顾一切，对世界尖锐，对亲人温柔。米米当上歌手以后，认识了许弋，两人相爱，却被蒋皎的经纪人害死……
　　夏吉吉：这个世上仅存的天使，在她的身上，有着其他人没有的宁静，温柔和包容。张漾曾经用一元钱买下她的画，从而铸就了一段缘分。她用她自己的平淡守护米米的精彩，米米死后，也是她主动宽释了一切仇恨。吉吉也爱着张漾，并在酒吧里救出了张漾。但是最后，他还是将失忆的张漾送回了小耳朵的身边，自己放手，云淡风轻。
　　“我早已经是没有了翅膀的鸽子，怎么还可以飞翔呢？”

{蒋皎（蒋雅希）}
　　许弋的初恋情人，后来爱上了张漾，因为张漾爱上了吧啦和小耳朵而痛苦不堪，性情大变。被父亲捧成歌星后，她不顾一切地抓住许弋，甚至不惜害死夏米米。被仇恨煎熬的她最后死于许弋制造的酒吧大火，这对她而言，未尝不是一种解脱。
　　PS：美丽富有的蒋皎其实很有资格骄傲。她的卑微，只是面对一份永远得不到的感情。
　　这个女生的身上，其实有巨大的能量。像一颗流星，与天空摩擦发出巨大的光。
　　“ 过去的过不去 明天不会远”
